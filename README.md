#  Sistema de Reservas de Canchas Deportivas

##  Descripción
Aplicación desarrollada en Java para gestionar clientes y reservas de canchas deportivas utilizando Programación Orientada a Objetos, HashSet y HashMap.

##  Objetivo
Administrar clientes y reservas mediante operaciones CRUD aplicando estructuras de datos eficientes.

##  Tecnologías utilizadas

- Java
- Programación Orientada a Objetos
- HashSet
- HashMap
- Interfaces
- Scanner

##  Estructura del proyecto

- Cliente.java
- CrudReserva.java
- ReservaCancha.java
- Test.java

##  Funcionalidades

- Registrar cliente
- Mostrar clientes
- Registrar reserva
- Actualizar reserva
- Eliminar reserva
- Buscar cliente
- Mostrar reservas
- Calcular total de horas reservadas
- Mostrar cliente con mayor reserva

##  Validaciones implementadas

- Cédula de 10 dígitos
- Edad entre 15 y 60 años
- Teléfono numérico
- Horas entre 1 y 5
- No permitir clientes duplicados
- No permitir reservas de clientes inexistentes

##  Estructuras utilizadas

### HashSet
Almacena clientes sin permitir duplicados.

### HashMap
Relaciona la cédula del cliente con las horas reservadas.
